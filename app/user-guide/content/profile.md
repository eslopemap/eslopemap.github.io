## Open the profile

The profile opens automatically when you import a track with at least two points. You can also reopen it at any time with the `Profile` button in the workspace toolbar.

![Elevation profile visible under the map after importing a track.](./assets/profile-panel.png)

*The profile panel stays connected to the active track and updates when the selection changes.*

## What the profile shows

The profile panel can display:

- elevation,
- track slope,
- terrain slope,
- speed,
- vertical speed,
- pause markers.

Use the gear menu in the profile header to turn each dataset on or off.

## Switch the X axis

The profile can be read against:

- distance,
- elapsed time,
- elapsed time without pauses,
- absolute date and time.

Time-based modes are most useful for imported GPX files that already contain timestamps.

## Link the chart back to the map

Hovering the chart highlights the matching point on the map. If the hovered point is off screen, Slope Mapper pans to bring it back into view.

If you have an active rectangle selection on the track, the profile narrows to that span and shows a reset control so you can return to the full line.

## When to use the profile

| Task | Why the profile helps |
| --- | --- |
| Compare terrain steepness along a route | Switch between track slope and terrain slope |
| Spot long stops in recorded GPX files | Enable pause markers and use a time-based X axis |
| Check whether a manual edit changed the climb or descent | Keep the profile open while editing |
